# Mashine Benchmarks

**Задача:** OCR (0-9 + A-Z, ~36 символов), 256 тайлов, 216 тактов

**Результат:** 36 решений, 0 коллизий, 1.27 ms на всю работу

## 📊 Сводная таблица

| Платформа | Среднее время такта (мкс) | Среднее время на выдачу решения (мкс) | 
| --------- | ------------------------- | ------------------------------------- |
| i5-3550 (2012) | 17.8 | 77.1 |
| Xeon E5 v4 (~2016) | 6.4 | 38.3 |
| EPYC 7003 (~2021) | 5.9 | 35.2 |

## Выводы

### Масштабирование: ×2 от i5 к серверу

Время цикла машины:

- i5-3550: 17.8 мкс
- Xeon E5: 6.4 мкс (×2.0)
- EPYC 7003: 5.9 мкс (×2.2)

Почти линейное ускорение при ×10 разнице в возрасте железа.

### Производительность: 12k - 30k FPS на 256 тайлах

- 12,970 FPS (i5) = 77.1 мкс на полный цикл выдачи решения (1-10 тактов, зависит от сложности паттерна)
- ~16 мкс на такт — отлично для интерпретируемого кода на CPU

Для сравнения:

- LLM инференс: ~10-100 мс на токен = 10-100 FPS
- Decima-8: 10,000+ FPS на рефлекс

> Не просто «быстрее» — «другой порядок величины».
