## 1. 简介

**Decima-8 提供第三条道路。**

我们不选择"慢速数字"和"噪声模拟"之间。我们用单时钟多比特激活构建模拟动力学的数字仿真。

**我们的假设：**

1. **Level16** — 激活值 0 到 15 单时钟传输，无需迭代累加
2. **中继激活替代分组路由** — 图块不互相传输数据，通过方向标志形成激活图
3. **两相周期 READ → WRITE** — 固定周期时间，无随机延迟

> *"我们不仿真神经元。我们构建识别即物理的织构。"*

## 2. 数学基础

### 2.1 Level16：语义四比特

```
thr_cur16 ∈ [0..15]  // 4 bits，一四比特
```

**一时钟 = 一值**。非频率，非模式，非"窗口内脉冲数"。

| 架构 | 编码 | 每值时钟数 |
|------|------|-----------|
| Loihi / SpiNNaker | spike ∈ {0, 1} | N 时钟 (频率) |
| Decima-8 | activation ∈ {0..15} | 1 时钟 |

### 2.2 SignedWeight5：带抑制的加权连接

```
weight ∈ [-7..+7]  // mag3 + sign1
```

**加权求和：**

```
row_raw_signed[r] = Σ (in16[i] × weight[r][i])  // i=0..7
范围：[-840..+840]
```

### 2.3 激活函数：双路径

**路径 A（累加器）：** `thr_cur16 += row_raw_signed[r]` (signed i16)

**路径 B（总线输出）：** `row16_out = clamp15((max(row_raw, 0) + 7) / 8)`

### 2.4 累加器 + Signed Decay

```
if (decay16 > 0) {
  if (thr_tmp > 0) thr_tmp = max(thr_tmp - decay16, 0)
  else if (thr_tmp < 0) thr_tmp = min(thr_tmp + decay16, 0)
  // ❗ 永不跳过 0!
}
```

### 2.5 范围熔丝

```
locked = 1, 如果 thr_cur16 ∈ [thr_lo16, thr_hi16]
```

## 3. 硬件架构

### 3.1 Conductor ↔ Island

**Conductor（数字指挥）：**
- 管理相位（EV_FLASH, EV_BAKE, EV_RESET_DOMAIN）
- 通过 SPI 加载权重
- 设置 VSB_INGRESS，读取 BUS16

**Island（模拟岛）：**
- 图块织构（8×32 到 32×128）
- VSB：8 路 Level16 输入
- BUS16：8 路诚实求和输出
- PATTERN_ID：专用通道

### 3.2 两相周期

```
┌─────────────┬──────────────┬─────────────┬─────────────┐
│ PHASE_READ  │ TURNAROUND   │ PHASE_WRITE │ READOUT     │
│ (0..8 μs)   │ (10..12 μs)  │ (12..18 μs) │ (18..20 μs) │
└─────────────┴──────────────┴─────────────┴─────────────┘
```

**固定延迟 20 μs**（i5-3550 仿真器）

### 3.3 中继激活（无路由器 NoC）

```
ACTIVE[t] = 1, 如果:
- t 有 BUS_R == 1 (源/根), 或
- ∃ 祖先 p: ACTIVE[p]==1 && locked_before[p]==1 && 有边 p→t
```

**0% 面积用于路由器，零抖动**

### 3.4 分支坍缩

```
if (ACTIVE[t] == 0) {
  thr_cur16 := 0
  locked := 0
  drive_vec := {0..0}
}
```

### 3.5 双注（Double Strait）

**BAKE_FLAG_DOUBLE_STRAIT** — 内部双时钟提高选择性

| 场景 | 无双注 | 双注 |
|------|--------|------|
| 输入"6"，检测器"6" | 触发（可能对"3"误触发） | 可靠触发（双倍激活） |
| 输入"3"，检测器"6" | 误触发 | 被拮抗剂阻断 |
